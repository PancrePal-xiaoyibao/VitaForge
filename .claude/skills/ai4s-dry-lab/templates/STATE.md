## 规则提醒（每次恢复时必读）

1. **W1**: 每次操作前写 WORKLOG — 覆盖：运行脚本、修改代码、参数调整、Bug修复、环境配置
2. **W2**: 每次产生输出写 README — 逐图逐表引用生成代码解释
3. **Gate**: 每 Phase 完成后运行检查（含 W1/W2 验证）
4. **Human-in-the-loop**: Phase 切换、意外发现、追加分析 → 必须与研究者确认
